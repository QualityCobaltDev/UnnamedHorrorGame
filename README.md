# UnnamedHorrorGame

PolygonForge and Mike Brogan - Horror Game Project on GoDot

## Main Menu Prototype

A small Python/Pygame script provides a fullscreen main menu with three options:

- **Start Game**: shows a placeholder game screen.
- **Settings**: displays a placeholder settings screen; press ESC to return.
- **Exit Game**: quits the application.

The window is created at the maximum resolution of the player's desktop in fullscreen mode.

### Running

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Start the prototype:
   ```bash
   python main.py
   ```

Press the arrow keys to move through options and Enter to select.
