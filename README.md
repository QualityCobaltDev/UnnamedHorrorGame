# The Talisman

Base filesystem for a Godot-powered 3D horror game.

## Features
- Procedurally generated forest map with cabins and foliage
- Monster that accelerates as the player collects talismans
- Three talismans hidden in randomized cabin locations
- Random jump scares: monster clones, noises and popups
- Retro inspired low-resolution visuals reminiscent of PS1 and Nintendo DS games

This repository now includes basic gameplay logic but still requires assets and
further polish.
